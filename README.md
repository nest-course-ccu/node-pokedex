# Run in develop

1. Clone the project
2. Run
```
npm i -g yarn
yarn install
```
## Install
3. Have Nest CLI
```
npm i -g @nestjs/cli
```

## Database
```
docker-compose up -d
```

## Stack 
* MongoDB
* Nest
